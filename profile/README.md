## Overview

Unarchiver Advanced is a free archive extraction utility that opens ZIP, RAR, 7z, TAR, GZ, BZ2, ISO, and dozens of other compressed and archived file formats that the built-in macOS archive utility cannot handle. It replaces the limited built-in extractor with comprehensive format support that handles virtually every archive format in common use and many obscure formats that users occasionally encounter. The application integrates with Finder as the default archive handler, making extraction seamless.

Format support covers over 40 archive formats including ZIP, RAR, 7z, TAR, GZ, BZ2, XZ, LZMA, ISO, CAB, MSI, EXE (self-extracting), RPM, DEB, NSIS, and many others that the built-in macOS extractor refuses to open. The breadth of format support means that virtually any compressed file downloaded from any source will extract without searching for a format-specific tool. Password-protected and multi-part archives are handled transparently.

Encoding handling solves the common problem of garbled filenames in archives created on Windows or Linux systems using different character encodings. The application detects the encoding used for filenames within the archive and converts them correctly to Unicode, ensuring that Japanese, Chinese, Korean, Cyrillic, and other non-Latin filenames display correctly after extraction.

Finder integration registers as the handler for all supported archive formats, enabling double-click extraction that works identically to the built-in ZIP handler. The integration is seamless — archives open as if the built-in extractor supported them natively. Extraction location, behavior after extraction, and other preferences are configurable.

For anyone who downloads compressed files and encounters formats that the built-in extractor cannot handle, Unarchiver Advanced is the extraction tool that opens everything with the simplicity of a double-click.
